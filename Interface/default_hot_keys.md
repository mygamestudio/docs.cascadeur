可以使用**快捷键**访问Cascadeur中的许多常用功能。此页面列出了每个默认快捷及其功能。

可以使用[**快捷键编辑器**](hotkey_editor.md)自定义快捷键，可通过从[主菜单](main_menu.md)中选择**Object**→**Hotkey Editor**进行访问。该编辑器还可用于为默认情况下不带功能的功能设置热键。


## 常规程序快捷键

下表列出了用于访问Cascadeur应用程序常规功能的快捷键

**命令**|**快捷键**
-|-
创建新场景|Ctrl + N
删除所选对象|Del
退出程序|Alt + F4
