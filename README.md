# 介绍

Cascadeur是一款无需动作捕捉即可创建角色动画的软件。使用基于物理学的方法，它可以为电影和视频游戏创建富有表现力和逼真的动画。Cascadeur目前处于测试阶段。您可以在我们的[网站](https://cascadeur.com/)上注册Beta测试。本手册旨在帮助您学习如何使用Cascadeur。在这里，您可以了解该软件为动画处理提供了哪些工具，以及如何使用它们。
