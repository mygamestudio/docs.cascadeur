# 轨道拉伸模式

此模式用于调整[**插值区间**](interpolation.md)。在此模式下编辑区间时，每个选中的关键帧都会移动并保持其相对于区间长度的位置。如此一来您可以修改一个动作的长度同时保持时间轴的结构。

1. 在时间轴上选择一个区间
2. 从**Edit**菜单中选择**Track stretching mode**（或**Ctrl + T**）
3. 现在您选择的区间将带有黄色框标记
4. 拖动区间的最后一帧以调整其长度

**注意：** 被调整的区间的插值类型将被改为[固定类型](interpolation.md/#固定插值)
