# Ubuntu

本页提供有关在**Ubuntu**操作系统上安装Cascadeur的说明。建议在安装之前确保系统满足[系统要求](SystemRequirements.md)。

## 安装
1. 下载包含程序文件的压缩文件
2. 下载完成后，使用任一压缩文件管理器打开压缩文件

压缩文件包含一个名为cascadeur-linux的目录。将其解压缩到硬盘驱动器上的任何位置。

此时安装已完成，可以启动程序。
